# CNN-MLP Hybrid Model for Dislocation Stress Field Prediction

This repository contains the code for generating synthetic dislocation configurations, extracting their physical properties (density, stress fields), and training a hybrid Convolutional Neural Network - Multi-Layer Perceptron (CNN-MLP) model to predict the internal stress fields.

## Project Structure

The pipeline consists of three main Python scripts:

1. **`single_thread_active_all.py` (Data Generation)**
   - Generates synthetic 3D dislocation configurations (in-plane rotated octagons) for FCC Copper.
   - Calculates the dislocation density fields and internal stress fields using the Mura equation.
   - Saves the generated configurations, tabular features (SSD, GND, etc.), and distributional data as compressed `.npz` files.
   - *Note:* This script is optimized to run on a single thread to allow for massive parallelization across cluster nodes.

2. **`density_extraction.py` (Physics & Computation Engine)**
   - A high-performance, Numba-accelerated module used by the data generator.
   - Computes continuous dislocation density tensors from discrete segments.
   - Calculates internal stress fields \(\tau_{int}\) using linear elasticity and Mura's formula.
   - Extracts physical features such as Statistically Stored Dislocations (SSD) and Geometrically Necessary Dislocations (GND).

3. **`CNN_MLP.py` (Machine Learning Pipeline)**
   - Loads the `.npz` data generated by the simulation.
   - Preprocesses the data using custom log transformations, `RobustScaler`, and `MaxAbsScaler`.
   - Optimizes model hyperparameters using **Optuna**.
   - Trains an ensemble of 5 hybrid neural networks. The model uses an MLP branch for tabular physical features and a 1D-CNN branch for angular dislocation density distributions.
   - Evaluates the ensemble and generates publication-ready plots (Accuracy, Residuals, Error CDF, Feature Importance).
